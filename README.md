\# 🏦 Finora - Smart Digital Banking Portal



\## 📌 Project Overview

Finora is a Java EE-based web application designed to simulate core banking operations in a digital environment. It provides secure and structured banking functionalities using MVC architecture.



This project demonstrates full-stack web development concepts using Java technologies such as JSP, Servlets, JDBC, and Java DB.



\---



\## 📸 Project Screenshots



\## 🚀 Features

\- User Registration and Login Authentication

\- Account Management

\- Deposit and Withdraw Funds

\- Fund Transfer between Accounts

\- Transaction History Tracking

\- Beneficiary Management

\- Loan Management System

\- Support Ticket System

\- Secure Session Handling



\---



\## 🛠️ Technologies Used

\- Java EE

\- JSP \& Servlets

\- JDBC

\- MVC Architecture

\- Java DB (Apache Derby)

\- HTML, CSS, Bootstrap

\- JavaScript



\---



\## 🧱 Architecture

\- Model-View-Controller (MVC)

\## ⚙️ How to Run

1\. Open project in NetBeans IDE

2\. Configure GlassFish Server

3\. Set up Java DB database

4\. Run project on server

5\. Open browser at localhost URL
